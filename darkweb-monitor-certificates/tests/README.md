# README for Tests

This directory contains unit tests for the Dark Web Monitor certificate system. The tests are designed to ensure the functionality and reliability of the application components, including certificate generation, storage, verification, and the admin dashboard.

## Running Tests

To run the tests, ensure you have the necessary dependencies installed as specified in the `requirements.txt` file. You can execute the tests using the following command:

```
pytest
```

## Test Files

- **test_certificates.py**: Contains tests for the certificate generation, storage, and verification functionalities.
- **test_admin.py**: Contains tests for the admin dashboard and performance tracking functionalities.

Make sure to run the tests regularly to maintain the integrity of the application as new features are added or changes are made.
